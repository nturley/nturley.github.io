Step 1: Tag Feedback Edges

  • If you ignore or reverse these edges it produces a directed, acyclical graph (DAG)
  • Which cycles to favor as feedback is domain specific
    • LabVIEW favors feedback nodes
    • Digital schematics favor flip flops
    • A lot of implementations try to tag as few as possible
  • Which edge is tagged as feedback will change how the diagram looks
